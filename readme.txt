Purpose
~~~~~~~

The main purpose of this website is to allow admin adding painting exhibtions, paintings; 
so that visitors can be able to register themselves to attend that exhibition.

admin
~~~~~

Admin can:
    -create other painters accounts
    -delete painter account
    -create exhibitition
    -delete exhibition
    -adding painting to given exhibition
    -change customer status
    -delete customer
    -see all paintings
    -delete any painting
    -add painting

painter
~~~~~~~
Painter can:
    -see his/her paintings
    -add new painting
    -delete painting

visitor
~~~~~~~
Visitor can:
    -see available exhibition
    -register themselves to attend exhibition
    -see galleries
    -like, share and buy a painting
